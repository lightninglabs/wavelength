# db/actordelivery/sqlc

## Purpose

Generated type-safe query layer for the durable actor mailbox schema
(mailboxes, outbox, ask results, FSM checkpoints, dead letters). Generated
by `sqlc` from `db/actordelivery/queries/mailbox.sql`; regenerate with
`make sqlc`. Do not edit by hand.

## Key Types

- `Queries` / `Querier` — generated query struct and interface (enqueue,
  lease, peek, ack/nack, extend, expire, outbox claim/complete/fail,
  dedup, FSM checkpoints, dead letters).
- `MailboxMessage`, `OutboxMessage`, `AskResult`, `FsmCheckpoint`,
  `DeadLetter`, `ProcessedMessage` — row models for the actor-delivery
  tables.

## Relationships

- **Depends on**: none (leaf generated package; only `database/sql`).
- **Depended on by**: `db/actordelivery` (wraps `Queries`/`Querier` in
  `ActorDeliveryQueries` / `BatchedActorDeliveryQueries`).

## Invariants

- Generated by `sqlc` — regenerate via `make sqlc`, never edit manually.
- Source of truth is `db/actordelivery/queries/mailbox.sql` and the
  migrations in `db/actordelivery/migrations`.
